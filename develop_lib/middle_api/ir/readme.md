# IR

## 麦克风IR对频协议
* 不采用标准的 NEC 协议

* 自定义帧
    帧头 + 命令码 1B + 命令码取反 1B + 数据 4B + 数据 4B

    1. 帧头
        5ms 低电平 + 5ms 高电平

    2. bit '0':
            2ms 低电平 + 1ms 高电平
    3. bit '1':
            2ms 低电平 + 2ms 高电平

    4. 命令码对应数据含义
        | command | data |
        |---------|------|
        |   0x01  | freq |
        |   0x02  |  vol |
        |   0x03  | tx_p |
